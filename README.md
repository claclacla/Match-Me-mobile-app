# Match me! (Mobile app)

The app has been initialized using Expo, and the user interface is built with UI Kitten.

## Initialization

```bash

npx create-expo-app MatchApp

cd MatchApp
npx expo install @ui-kitten/components @eva-design/eva react-native-svg

```

## Development

```bash

npx expo start
```
